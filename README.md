# これは何？
iOSアプリの勉強用に作った、自分専用イラスト閲覧アプリのプロトタイプです。 ChatGPTに解説してもらいながら機能を追加していってます。まあぶっちゃけ気づいたらほとんどコピペしてるけど、コード生成ではなく、実装方針を相談しながら基本は写経しつつ進めているので「これはバイブコーディングとは違うんだ！」と念じながら……
どうみてもバイブコーディングです。本当にありがとうござ

# どうやって使うの？
フォルダに表示させたいイラストをまとめて、pythonスクリプトを走らせてファイルリストjsonファイルを生成。それをシードにイラストを読み込んで、タグや表示順をゴニョゴニョします。

# どんな機能があるの？
## タグ管理機能
- タグの追加・編集
- 各イラストに対するタグ付け
- AND検索（複数タグ一致）
- 除外タグフィルタ
- 未タグのみ表示
タグデータは tags.json として Documents に永続化。

## 自動ソート機能（動的ランキング）
表示順は以下の優先順位で決定：
- 表示スコア（閲覧数 + お気に入りボーナス）
- 最終閲覧日時
- 元のJSON順
タグの0番目を「お気に入り」として扱い、スコアに +5 の重み付け。
